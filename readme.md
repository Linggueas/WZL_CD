# Qt 云存储客户端 + 服务器（网盘系统）

一款基于 Qt 框架开发的跨平台网盘系统，提供完整的文件/文件夹云端存储、管理及传输功能。服务端与客户端均为 Qt 实现，采用自定义 TCP 协议通信，支持多客户端并发访问。

## &#x20;功能特性

### 客户端

*   用户注册/登录（配合 MySQL 记录在线状态）
*   文件树形目录展示（支持文件夹嵌套）
*   上传文件、下载文件、新建文件夹、重命名、删除
*   文件传输进度条显示
*   右键菜单快捷操作
*   自动与服务端保持心跳连接

### 服务端

*   基于 `QTcpServer` 的并发多线程架构
*   自定义 TCP 协议（报文 ID + 数据长度 + JSON 体）
*   粘包处理与分帧解析
*   客户端连接状态管理
*   操作日志记录（本地文件或数据库）
*   MySQL 用户状态持久化
